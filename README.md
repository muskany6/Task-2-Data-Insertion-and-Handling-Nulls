# Task-2-Data-Insertion-and-Handling-Nulls

 Objective
The objective of this task is to practice SQL data manipulation by performing INSERT, UPDATE, and DELETE operations while handling NULL values appropriately.

Tools Used
- Oracle FreeSQL

Table Used
Student

 Operations Performed

 1. INSERT
- Inserted multiple student records into the `Student` table.
- Added values for Student ID, Name, Age, Marks, and Department.

2. Handling NULL Values
- Used NULL/default values where required to understand how SQL handles missing data.

3. UPDATE
- Updated the department of a specific student.
- Updated the marks of another student using the `WHERE` clause.

4. DELETE
- Deleted a specific student record using the `WHERE` clause.

5. SELECT
- Retrieved the final data from the table using `SELECT`.
- Displayed the records in ascending order of Student ID using `ORDER BY`.

SQL Concepts Covered
- INSERT INTO
- UPDATE
- DELETE
- WHERE Clause
- ORDER BY
- NULL Handling

Outcome
Successfully performed data insertion, updating, deletion, and retrieval operations on the Student table. Learned how to modify specific records using the `WHERE` clause and maintain clean and consistent data.
